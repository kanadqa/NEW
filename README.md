# Учет расходов и доходов

Обновленный офлайн-трекер в формате «приложения» с боковым меню и переключаемыми экранами. Данные сохраняются в браузере, есть диаграммы и ручное управление категориями и подкатегориями.

## Что нового

- Навигация по разделам: «Обзор», «Операции», «Категории», «Отчеты».
- Три варианта интерфейса (Комфорт/Баланс/Компакт) — выбираются в боковом меню.
- Единая форма добавления: категория и подкатегория в одном действии.
- Полноценное управление категориями: переименование, удаление, перенос подкатегорий.
- Добавлены круговые диаграммы доходов и расходов и график динамики по дням.
- Категории и подкатегории можно перетаскивать между карточками мышью.
- Кнопка «Отменить» возвращает последнее действие.

## Как пользоваться

1. Откройте `index.html` в браузере.
2. Перейдите в «Операции» и добавьте запись.
3. В разделе «Категории» добавляйте категории и подкатегории через один блок.
4. Перетаскивайте подкатегории в другие карточки.
5. Чтобы сделать подкатегорию отдельной категорией, перетащите ее в блок «Категории верхнего уровня».
6. Чтобы превратить категорию в подкатегорию, перетащите карточку категории в другую категорию.
7. В «Обзоре» смотрите диаграммы и график доходов/расходов по дням.
8. Нажмите «Экспорт CSV», чтобы сохранить данные для Excel/Google Sheets.

## Категории и подкатегории

Категории и подкатегории задаются вручную в блоке «Категории». Тип категории определяет, в каком режиме операции она доступна (доходы или расходы).

## XLSX-альтернатива

Если вам удобнее работать именно в Excel, используйте файл `template.csv` и откройте его в Excel как CSV. Затем сохраните как `.xlsx` при необходимости.

## Запуск локального сервера (необязательно)

Если браузер блокирует доступ к локальному хранилищу при открытии файла напрямую, запустите простой сервер:

```bash
python -m http.server 8000
```

Затем откройте `http://localhost:8000`.
